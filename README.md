# Chore Safari (チョア・サファリ)

**「現実のお手伝いと大きな成長が、デジタルの冒険に繋がる。」**
家族3人の子供たち（生年月日から動的に年齢・役割を算出）が協力して遊ぶ、お手伝い連動型の知育＆どうぶつ捕獲Webプラットフォーム。

## 概要
現実世界での「お手伝い」や「テストの点数・習い事の昇級」を親が承認することで、子供たちにゲーム内コインが付与されます。そのコインを使ってクレーンゲームで罠や餌を獲得し、森で動物（や恐竜）を捕まえて「家族共通の図鑑」を完成させるのが目的です。

## コアシステム（3C）
1. **Coin & Bank (経済基盤)**
   * 子供ごとの個別コイン残高管理
   * 親用スマホからの報酬付与（お手伝い、特大ボーナス）＆ペナルティ没収
2. **Crane & Craft (準備と在庫)**
   * 家族共有のインベントリ（WMS的な在庫管理）
   * 獲得した素材を組み合わせるパズル的な罠クラフト
3. **Catch & Compete (メインゲーム)**
   * 条件（罠＋餌）を満たして動物を捕獲する探索機能
   * 捕まえた動物たちで競う熱血レース機能

## 技術スタック
* **Frontend / Backend:** Next.js (App Router), TypeScript, Tailwind CSS
* **Database:** PostgreSQL (Prisma ORM)
* **Mini-game Engine (予定):** Phaser.js (クレーンゲームやレースの描画)
* **AI Integration (予定):** LLM API (レース実況生成、トンデモ罠の自動判定など)
* **Infrastructure:** ローカルPC上のDocker (docker-compose) + Cloudflare Tunnels (外部アクセス用)
